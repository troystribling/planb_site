has_ancestor

has_ancestor is a Ruby on Rails plug-in that provides an implementation of Class Table Inheritance based on Rails Polymorphic Associations. The plugin adds two methods to ActiveRecord::Base, has_ancestor and has_descendants. With has_descendants a model declares that its attributes and methods are inheritable. With has_ancestor a model declares that it inherits the attributes and methods of the specified model. Once ancestor and descendant relations are declared the models interact as expected from class inheritance, this includes defining methods that behave in a polymorphic manner. See Construct Model Abstractions with has_ancestor for a discussion of a simple implementation using the plugin and has_ancestor finders for a discussion of database query construction.

acts_as_aln

acts_as_aln, or Acts as Abstract Layered Network, is a Ruby on Rails plug-in that provides an infrastructure for construction of enterprise information models. The Abstract Layered Network model is inspired by the network information models described in ITU M.3100, TMF 814 and Telecordia GR-1093-CORE that attempt to develop detailed object models that describe many of the attributes and state transitions of communication networks. In this vein the DMTF Common Information Model attempts to develop a model of the enterprise infrastructure. Of the previously stated references only DMTF has documentation in the public domain. Instead following the path pursued by these efforts acts_as_aln only considers a few objects, the relations between the objects and a couple of attribute types, namely, the information that most strongly impacts the structure of an application built on the model. This greatly simplifies the information model and extends its domain of applicability. While the other models are limited to communications networks, and enterprise hardware and software in the case of DMTF-CIM because of their detail, acts_as_aln can be applied to these domains and possibly to seemingly unrelated domains such as project management, customer relationship management, human resource management, business intelligence and financial management.

linux agent

linux agent is an implementation of acts_as_aln applied to Red Hat and Debian Linux dialects. Linux Agent provides the ability to discover system resources and configuration and report system performance using SYSSTAT.
